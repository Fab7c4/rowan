hamilton
========

Welcome to the documentation for hamilton, a pure Python software package for working with the various formalisms for rotation in 3D.
There are a number of ways to represent these formalisms, an issue that hamilton aims to solve.

To install hamilton, first clone the repository `from source <https://bitbucket.org/glotzer/hamilton>`_.
Once installed, the package can be installed using setuptools::

    $ python setup.py install --user
